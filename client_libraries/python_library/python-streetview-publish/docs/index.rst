GAPIC library for the Street View Publish API
=============================================================================================================

This is the API documentation for ``gapic-google-maps-streetview_publish-v1``.

gapic-google-maps-streetview_publish-v1 uses google-gax_ (Google API extensions) to provide an
easy-to-use client library for the `Street View Publish API`_ (v1) defined in the googleapis_ git repository


.. _`google-gax`: https://github.com/googleapis/gax-python
.. _`googleapis`: https://github.com/googleapis/googleapis/tree/master/google/streetview/publish/v1
.. _`Street View Publish API`: https://developers.google.com/apis-explorer/?hl=en_US#p/streetviewpublish/v1/
